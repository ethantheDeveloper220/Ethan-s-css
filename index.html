Rainbow Transparent

A colorful animated rainbow theme for Spicetify with translucent UI.

Download

How to Install the Rainbow Transparent Spotify Theme

  1. Download the theme by clicking the button above.
  2. Extract the files to your .spicetify/Themes directory.
  3. Open PowerShell and run the following commands:
    4. spicetify config current_theme RainbowTransparent
    spicetify apply
  4. Your Spotify theme should now be updated with the new Rainbow Transparent look!

Spotify Theme

Rainbow Transparent

A colorful animated rainbow theme for Spicetify with translucent UI.

Download

How to Install the Rainbow Transparent Spotify Theme

  1. Download the theme by clicking the button above.
  2. Extract the files to your .spicetify/Themes directory.
  3. Open PowerShell and run the following commands:
    4. spicetify config current_theme RainbowTransparent
    spicetify apply
  4. Your Spotify theme should now be updated with the new Rainbow Transparent look!

A vibrant, rainbow-themed design for Nitro Type users. Fun and colorful UI enhancements!

Download

How to Install the Nitro Type Rainbow V2.5 Theme

  1. Download the theme by clicking the button above.
  2. Go to your userstyles folder (or wherever your Nitro Type theme files are located).
  3. Place the downloaded CSS file into the theme directory.
  4. Refresh or restart Nitro Type to see the vibrant new rainbow theme in action!
